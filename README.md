# База данных для музыкального сервиса

## Описание
Проект включает схему базы данных для музыкального streaming-сервиса.

## Файлы
- `Схема.png` - диаграмма схемы БД
- `Музыкальный_сервис.sql` - SQL-запросы для создания БД

## Особенности
- Связь многие-ко-многим между исполнителями и жанрами
- Совместные альбомы нескольких исполнителей
- Система сборников с треками из разных альбомов
